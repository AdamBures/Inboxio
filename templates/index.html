FAQ

What is Inboxio ?

Inboxio is a file saving website that compresses your files and stores them in a best quality as possible.


Why was Inboxio made ?

Inboxio was made as a hackathon project for Steel City Hacks 2023.


What is the goal of this project ?

The goal of this project is to provide a best file compression on the market. Another goal is to learn more about Web Development and File Compression.
